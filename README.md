## ğŸ“¦ Installation

### Prerequisites

- Node.js (v16 or higher)
- npm

### Local Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/tnpaul/MarkdownPreviewer.git
   cd MarkdownPreviewer
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Install TypeScript types for react-syntax-highlighter**
   ```bash
   npm install --save-dev @types/react-syntax-highlighter
   ```

## ğŸ¯ Usage

1. **Write Markdown** - Type or paste Markdown content in the left editor pane
2. **View Preview** - See the rendered output in real-time on the right
3. **Resize Panes** - Drag the vertical divider to adjust pane sizes
4. **Toggle Sync Scroll** - Enable/disable synchronized scrolling between editor and preview
5. **Switch Theme** - Toggle between light and dark mode
6. **Copy Content** - Click the Copy button to copy your Markdown to clipboard
7. **Reset Editor** - Click Reset to clear all content
